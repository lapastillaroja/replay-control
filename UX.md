# Companion App вҖ” UX Design

## Visual Style

**Clean & modern** вҖ” minimal, card-based layout, smooth animations. App store / Steam library feel.

## Navigation

**Bottom tab bar** (4 tabs) + **top bar** (search + favorites shortcut + settings gear)

```
в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”¬в”Җв”Җв”Җв”Җв”Җв”Җв”¬в”Җв”Җв”Җв”Җв”Җв”Җв”¬в”Җв”Җв”Җв”Җв”Җв”Җв”җ
в”Ӯ Home в”ӮGames в”Ӯ Favs в”Ӯ More в”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”Җв”ҙв”Җв”Җв”Җв”Җв”Җв”Җв”ҙв”Җв”Җв”Җв”Җв”Җв”Җв”ҙв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
```

- **Home** вҖ” dashboard: last played (hero card), recently played (horizontal scroll), stats
- **Games** вҖ” system cards grid вҶ’ tap system вҶ’ compact game list
- **Favs** вҖ” favorites list (flat or grouped by system)
- **More** вҖ” upload ROMs, backup, settings, Wi-Fi/NFS config, system info

On desktop, bottom tabs can become a **left sidebar**.
