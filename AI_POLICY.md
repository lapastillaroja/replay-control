# AI Policy

## For Users

This project was built with significant AI assistance. The developer reviews, understands, tests, and maintains all code.

## For Contributors

Contributions that use AI tools are welcome, provided:

- You understand and can explain the code you submit
- You disclose AI tool usage in your PR description
- You have tested your changes
- Comments and documentation are human-authored or human-reviewed

Inspired by [EFF's policy on LLM-assisted contributions](https://www.eff.org/deeplinks/2026/02/effs-policy-llm-assisted-contributions-our-open-source-projects) and [Ghostty's approach](https://github.com/ghostty-org/ghostty/blob/main/AI_POLICY.md).
