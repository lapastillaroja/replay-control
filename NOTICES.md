# Third-Party Notices

## Embedded Data (compiled into binary at build time)

### No-Intro DATs
ROM identification databases.
Source: [libretro-database](https://github.com/libretro/libretro-database)

### TheGamesDB
Game metadata (year, genre, developer, publisher, players, coop, rating).
Source: [TheGamesDB](https://thegamesdb.net/)
License: GPLv3 (codebase). Developer/publisher/genre name mappings fetched via API at build time, compiled into PHF maps. Raw API data not redistributed.

### MAME / FBNeo
Arcade game databases.
Source: [libretro-database](https://github.com/libretro/libretro-database)
License: MIT / MAME License

### Wikidata
Game series relationships.
Source: [Wikidata](https://www.wikidata.org/)
License: CC0 (public domain)
