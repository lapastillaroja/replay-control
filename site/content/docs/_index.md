Replay Control is a web-based companion app for [RePlayOS](https://www.replayos.com/), a retro gaming operating system for the Raspberry Pi. It runs as a local server on the Pi and gives you a browser UI for managing your game library, enriching it with metadata and artwork, and launching games on the TV — all from your phone, tablet, or desktop.

New here? Start with the [Getting Started](features/getting-started/) guide to install Replay Control on your Pi and get up and running in minutes.
